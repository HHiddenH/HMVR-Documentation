Start Guide

Installation

Just download the HMVR plugin from Epic Games Laucher and open the Demo project. If you bought plugin in Arstation you need to place Plugin into UE\Engine\Plugins\Runtime or your project folder into Project/Plugins.

Demo Level

HMVR includes free demo which you can try before buing the plugin and give the feadback on our discord server. Map is located at Plugin folder Content/Demo/Maps/Map_0

Player setup

For player setup we need 3 classes: AVR_PlayerPawn, AVR_MotionController, VR_PlayerPawn, VR_Inventory.

VR_PlayerPawn

Create blueprint from AVR_PlayerPawn class.
Here we can adjust some parametrs and choose AVR_MotionController class which we want to use.

VR_MotionController

Create blueprint from VR_MotionController class.
Here we need to chose our hand mesh(For FakeHandMesh and RealHandMesh. Fake one doesn't have a collision and can be used in the main menu) and an animation blueprint to make it work. To set up your own AnimBP look at Content/Assets/models/VR_Player/Hand/ABP_VR_RobotHand there you can find basic setup. Also don't forget to setup line traces properly for IK.

VR_Inventory

Create blueprint from AVR_Inventory class as we did before.
Here we can adjust some variables.

Grabbing

AVR_PickUpObject is an object which player can pick up using one of grab methods (enum EGrabMethod)

Weapons

AVR_WeaponBase class is used for easy weapon creation.
Setup is really easy just fill required varialbes and select skeletal mesh with animBP.

Interactables

Interactables are objects with which player can interact. Set up for the interactables objects are simple like weapon setup. Just fill the parametrs.
