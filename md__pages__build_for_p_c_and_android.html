General information

There is an option to use any VR standart. There is a plugin switcher button which allows developer to switch between VR plugins(SteamVR/Oculus/OpenXR). For building project under Android use Oculus or OpenXR.
After conversion user need to recompile the C++ project and reopen it. Visual Studio project must be regenerated. If project was converted to Oculus or OpenXR platform, it is recomended to make sure that SteamVR plugin is disabled. Otherwise developer might face packaging issues.
Note: To recompile plugin simply move it to the ProjectFolder/Plugins/. Then recompile the C++ project or not closing the editor go to Plugins and find Head Mounted VR. Then press the package button.
