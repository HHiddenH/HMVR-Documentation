HMVR 1.0
HMVR

Description:

This plugin integrates Head Mounted Software's HMVR features in an intuitive and easy to use way for Unreal developers. It implements physics base Player Pawn, physically based hands and a lot of interactive objects to play with!

Features:

  • Physics-based player pawn
  • Physical hands with finger collision and IK
  • Grabing variations
  • Doors and drawers
  • Weapon base
  • Equipment base
  • Levers and valves
  • Buttons
  • Inventory

Optimized Motion Controllers:

-Oculus Touch -Valve Index

Multiplayer Support: No

Supported Platforms: Windows 10-11
